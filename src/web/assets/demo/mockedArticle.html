Parseit

Приложение запущено в демо режиме и настоящий парсинг страниц отключен. Кстати, просто так получить html код внешней страницы из JavaScript не получится из-за CORS. Для этого я написал небольшой прокси-сервер на Express. В демо режиме он также недоступен, но при желании Parseit может полноценно работать в браузере.
